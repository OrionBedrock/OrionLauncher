## Visual refresh *(focus of this release)*

* **Multi-language UI:** switch between **English** and **Portuguese (Brazil)** at runtime; strings ship as JSON under `I18n/` and drive the whole shell, launcher screens, and localized Hub news files.
* **GitHub:** button in the **top-right** of the window opens `https://github.com/OrionBedrock/OrionLauncher`.
* **Updating existing instances:** in **Instance settings**, upgrade Bedrock to the **latest build in the same channel** (stable vs preview) when an update is available from the catalog.
* Reformulated launcher shell: dark sidebar navigation (Hub, Instances, Mods), profile strip at the top, and settings at the bottom.
* Hub hero area uses the wallpaper full-bleed behind content; **OrionBE logo** (`Assets/logo.png`) is shown **centered at the top** with proportional scaling.
* News / changelog panel remains below the Play row with Markdown rendering.

## Profile

* **Local launcher profile** replaces Microsoft / Xbox sign-in: nickname (short), optional **message** under the nickname (character limit enforced in the editor), and **custom avatar** copied into `~/OrionBE/cache`.
* Profile editor screen (avatar preview, pick/remove photo, save / back).

## Technical

* Sidebar and profile avatars load via **Avalonia `Bitmap`** from disk (`Stretch.Uniform` inside rounded clips) for reliable display on Linux.
* Removed MSAL / Microsoft Identity Client and related OAuth-only code paths.

## Repository migration *(temporary)*

* The project is undergoing a **gradual migration** of **folder layout** and **code organization**. Until that work settles, **some areas may look inconsistent or confusing**—thanks for bearing with us.
