## CORE CONCEPT

This application is a Minecraft Bedrock launcher and mod manager.

It is NOT meant to import existing installations.
It installs and manages everything from scratch.

All launcher data must be stored in:

/home/{username}/OrionBE

---

## FOLDER STRUCTURE (IMPORTANT)

Design a system that uses this structure:

/OrionBE
├── instances/
│    └── {instanceName}/
│         ├── config.json
│         ├── game/
│         └── mods/
├── mods/
│    └── {modName-version}/
│         └── mod.json
├── cache/
└── assets/

* Mods are GLOBAL (stored once)
* When installing a mod into an instance, it MUST be COPIED into instance `/mods/`
* Do NOT reference mods, always duplicate them
